# ID INFORMÁTICA LTDA

Site institucional e loja virtual da **ID INFORMÁTICA LTDA**, empresa localizada em **Cruzeiro do Sul - Acre**, com atuação em **informática e papelaria**.

O projeto foi pensado para apresentar a empresa, divulgar serviços, exibir produtos e permitir solicitação de orçamento.

## Sobre o projeto

Este site foi desenvolvido para atender duas necessidades principais:

- **Vitrine institucional**: apresentação da empresa, serviços e contato
- **Loja virtual**: exibição de produtos de informática e papelaria com possibilidade de compra online

## Funcionalidades

- Página inicial com apresentação da empresa
- Seção "Sobre nós"
- Lista de serviços de informática
- Catálogo de produtos
- Formulário de solicitação de orçamento
- Integração com WhatsApp
- Layout responsivo para celular e computador
- Identidade visual com cores roxo, azul e preto

## Como usar

### Opção 1: abrir localmente
1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador

### Opção 2: publicar em um servidor
1. Envie os arquivos para sua hospedagem
2. Configure o domínio
3. Certifique-se de que o arquivo principal seja `index.html`
